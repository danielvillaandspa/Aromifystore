מוצרים מובחרים להפצת ריח מקטאר – שמנים אתריים בלבד ודפיוזרי רסס חכמים.
המכשירים שלנו מפיצים ניחוח נפלא ומכסים שטחים מ-40 מ"ר ועד 500 מ"ר, ומתאימים באופן מושלם לחדרי רחצה, רכבים וחללים גדולים.
אנו מציעים מגוון רחב של סוגי ריחות איכותיים. המכשירים חכמים במיוחד וניתנים לשליטה מלאה ונוחה ישירות מהטלפון הנייד (סמארטפון).
